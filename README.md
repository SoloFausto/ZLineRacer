# Comet riders

This is a tron style line racer inspired by Armagetron. Currently there is support for having split-screen multiplayer, along with a basic gameplay implementation.

Project roadmap is maybe in the future a multiplayer mode, along with further improvements to the ship speed through player testing.

Sources for the assets are:
https://assetstore.unity.com/packages/2d/textures-materials/galaxia-2d-space-shooter-sprite-pack-1-64944
https://deep-fold.itch.io/space-background-generator

Done for the COSC481 -- Playful Thinking, Serious Coding class midterm by Fausto De Leon.

Generative AI was used in some minor ways to rework some code and just to think through stuff, and in some major ways when implementing
the grinding particles. When applicable, there are comments throughout the code pointing out the different sources of stuff.

Created with raylib and python.
