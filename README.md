# Comet riders
## Game Description:
Comet Riders is a fast-paced tactical line racer, where the main goal is to manouver around your rivals and get them to crash into your wall. 
Through local multiplayer, the game is focused around Player versus Player, where quick reflexes and predicting what the other player is going to do are keys to being the best at Comet Riders.

## Key Features:

### Fully Featured Split-Screen Multiplayer: 
The game features having up to 4 players in a single match, with the screen splitting up to four ways to allow each player to have its own separate screen.

### Risk and reward gameplay: 
With the mechanic of grinding, a player can gain speed by moving along-side a wall. This allows players to catch up to one another, while also allowing a risk-reward style of gameplay.

### Easy to learn, hard to master: 
The game features certain quality of life improvements that make the game accessible to everyone. Firstly, there is the pillow mechanic which cushions certain player impacts, giving players sufficient time to react. Secondly there are the radar arrows, which point to the location of other players and make the game more engaging. These both features make the game more accesible to players of all skill levels.

### Competitive graphics and inmersive sound:
The game features layering of various sound sources to have both music, spaceship motors, grinding noises, and player explosions at the same time, creating an inmersive experience for the game. The game also has competitive graphics that prioritize readibility above just graphical spectacle.
